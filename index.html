MySQL Workbench × Python データ連携
開発中

MySQL Workbenchで作成したテーブルに対して、Pythonからデータベースへ接続し、データを取得・表示するプログラムを開発中です。現在は接続して内容を表示する段階まで完了。今後はデータの追加・更新・削除（CRUD操作）まで拡張していく予定です。

Python MySQL mysql-connector-python
db_connect.py 
import mysql.connector

# MySQL Workbenchで作成したデータベースへ接続
conn = mysql.connector.connect(
    host="localhost",
    user="root",
    password="********",
    database="sensor_db"
)

cursor = conn.cursor()
cursor.execute("SELECT * FROM sensor_data")

for row in cursor.fetchall():
    print(row)

# 次のステップ: INSERT / UPDATE / DELETE に対応予定
cursor.close()
conn.close()
